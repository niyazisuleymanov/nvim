**REQUIREMENT**: You need to be running Neovim v0.5.0+

# Install

```console
git clone --depth 1 https://github.com/niyazisuleymanov/nvim \
  ~/.config/nvim
sh ~/.config/nvim/install.sh
```

# Update

```console
git -C ~/.config/nvim pull   # or go to ~/.config/nvim and git pull
sh ~/.config/nvim/install.sh
```
